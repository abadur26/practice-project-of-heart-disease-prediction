# Heart Disease Prediction

## Overview
This project uses machine learning to predict the likelihood of heart disease based on patient health parameters such as age, cholesterol, blood pressure, and more.

## Features
- Web-based interface for inputting health data
- Predictive model trained on medical datasets
- User-friendly design for quick results
